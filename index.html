  • LOGIN - FORMULÁRIO

    O site apresenta um formulário de login simples, e um formulário interativo, utilizando HTML e CSS. Ele contém campos para o usuário inserir seu e-mail e senha, e um botão para submeter as informações. O design é minimalista e tem um toque de humor, focado na funcionalidade básica de um formulário de autenticação.

  • CURRÍCULO

    O repositório no GitHub contém um projeto de currículo, utilizando HTML e CSS. Ele apresenta um modelo de currículo estilizado, incluindo código-fonte e uma estrutura básica de arquivos. O objetivo do projeto é servir como um exemplo prático de como criar um currículo digital com técnicas básicas de web design.
